# 이곡어때 - 노래방 추천 앱

노래방에서 지금 상황에 딱 맞는 노래를 추천해주는 웹 앱입니다.

## 📌 프로젝트 개요

### 서비스 목적
- 노래 실력과 무관하게 누구나 부담 없이 곡 선택 가능
- 분위기를 망치지 않는 곡 선택 지원
- CSV 데이터 기반 정확한 추천

### 핵심 기능 (2026.01.31 업데이트)
6단계 선택 → 맞춤 추천:
1. **나이** - 10대, 20대, 30대, 40대, 50대+, 상관없음
2. **가수 성별** - 남자, 여자, 상관없음
3. **장르** - 발라드, 힙합, 댄스, 락, 트로트, R&B, 상관없음
4. **분위기** - 잔잔, 달달, 신나는, 우울한, 애절한, 상관없음
5. **인원수** - 혼자, 듀엣, 그룹, 상관없음
6. **상황** - 감성충전, 분위기 띄우기, 데이트, 고음어필, 상관없음

→ **5곡 추천** (조건 필터링 + 랜덤)

## 🛠️ 기술 스택

- **Vite** - 번들러 (^6.0.0)

- **Google Fonts** - Noto Sans KR
- **Apps In Toss** - 웹 프레임워크 (선택적)

## ⚙️ 추천 알고리즘 (recommend.js)

```javascript
recommend(userSelection) {
    1. 모든 조건 AND 필터링
    2. 조건 부족 시 핵심 조건만 필터링 (분위기, 상황)
    3. 그래도 부족 시 전체에서 랜덤
    4. Fisher-Yates 셔플
    5. 5곡 반환
}
```

**핵심**: `null` 값은 해당 조건 무시 (상관없음)
