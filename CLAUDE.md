# 이곡어때 - 노래방 추천 앱

노래방에서 지금 상황에 딱 맞는 노래를 추천해주는 웹 앱입니다.

## 📌 프로젝트 개요

### 서비스 목적
- 노래 실력과 무관하게 누구나 부담 없이 곡 선택 가능
- 분위기를 망치지 않는 곡 선택 지원
- CSV 데이터 기반 정확한 추천

### 핵심 기능 (2026.01.31 업데이트)

**1. 프로필 입력**
- 유저 성별, 연령대, 방문 빈도, 선호 장르

**2. 3단계 선택 → 맞춤 추천**
- 화면1: 나이 + 가수성별
- 화면2: 장르 + 분위기
- 화면3: 인원수 + 상황

**3. 추천 결과**
- 5곡 추천 (조건 필터링 + 랜덤)
- 후기 작성 및 저장 (localStorage)

## ⚙️ 추천 알고리즘 (recommend.js)

3단계 폴백 전략:

```javascript
recommend(userSelection) {
    1. 모든 조건 AND 필터링
    2. 조건 부족 시 핵심 조건만 필터링 (분위기, 상황)
    3. 그래도 부족 시 전체에서 랜덤
    4. Fisher-Yates 셔플
    5. 5곡 반환
}
```

**핵심**: `null` 값은 해당 조건 무시 (상관없음)

자세한 내용은 [recommend.md](./recommend.md) 참고

## 💬 후기 기능

- **저장**: localStorage에 후기 저장
- **표시**: 날짜, 내용, 선택 조건 함께 저장
- **삭제**: 개별 후기 삭제 가능
